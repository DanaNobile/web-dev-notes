# Create your first Node App
 
* Create folder
* Inside create `app.js`

`app.js`

```
console.log('Hello World');
```

* **note** `console.log()` is not part of JavaScript programming language, it is always part of the runtime, whether it comes from a browser like Chrome or Node.js

## Where do the Node.js docs live?
* [node docs](https://nodejs.org/en/docs/)
* In sidebar choose the version of the Node API you are working with
* Search for `console` and here are [its docs](https://nodejs.org/dist/latest-v13.x/docs/api/console.html)

## Run app
* In project directory root folder
* Run

`$ node app.js`

* `Hello World` will print to the screen
